# PS Deals v3.5.1 — Pulido y productividad

## Funciones nuevas
- Buscador global con Ctrl+K o botón flotante de lupa.
- Navegación rápida con teclado.
- Búsqueda de módulos, productos y clientes.
- Toasts globales disponibles para todos los módulos.
- Indicador y advertencia de cambios pendientes.
- Duplicación de productos desde Inventario.
- Bloqueo temporal de doble registro de ventas.

## Pruebas recomendadas
1. Presionar Ctrl+K y buscar un módulo, producto y cliente.
2. Probar F2, F4, F7, F8 y F9 fuera de campos de texto.
3. Editar un formulario y comprobar el aviso de cambios pendientes.
4. Duplicar un producto y guardar la copia con otro SKU.
5. Registrar una venta y hacer doble clic para confirmar que solo se crea una operación.
