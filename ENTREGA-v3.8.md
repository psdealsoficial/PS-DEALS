# PS Deals v3.8 — Importación y exportación avanzada

## Funciones nuevas

- Exportación CSV para productos, clientes, ventas, compras, gastos y apartados.
- Archivos compatibles con Excel y codificación UTF-8.
- Importación de productos y clientes desde CSV.
- Detección automática de coma, punto y coma o tabulador.
- Vista previa de hasta 20 registros antes de guardar.
- Modos **Combinar** y **Reemplazar**.
- Coincidencias de productos por SKU, código de barras, ID o nombre.
- Coincidencias de clientes por teléfono, correo, ID o nombre.

## Prueba recomendada

1. Configuración → Intercambio avanzado.
2. Exporta Inventario en CSV.
3. Abre el archivo con Excel o LibreOffice Calc.
4. Cambia el stock de un producto y guarda como CSV.
5. Importa el archivo en modo Combinar.
6. Revisa la vista previa y aplica.
7. Abre Inventario y confirma el cambio.

Haz un respaldo JSON completo antes de probar el modo Reemplazar.
