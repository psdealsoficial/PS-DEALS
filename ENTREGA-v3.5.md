# PS Deals v3.5 — Dashboard Inteligente

Esta versión transforma el Dashboard en una herramienta de decisión usando los datos existentes en localStorage.

## Pruebas sugeridas
1. Abrir Dashboard y comprobar los tres comparativos.
2. Confirmar que “¿Qué debería hacer hoy?” enlaza al módulo correcto.
3. Registrar una venta y recargar el Dashboard.
4. Reducir el stock de un producto hasta su mínimo y verificar la prioridad.
5. Confirmar el semáforo de flujo, margen, inventario y cobranza.
